
## Getting Started
***
#### Update project
`mvn dependency:resolve`

#### Test
`mvn test`

#### Integration Test
`mvn integration-test`

#### Run the project
`mvn jetty:run`

#### Response Status
1. 200 - sucess
2. 404 - resource not found
3. 400 - bad request
4. 201 - created
5. 401 - unauthorized
6. 500 - server error
